# 摄影爱好者APP设计文档

## 概述

**项目名称**：摄影爱好者APP（HarmonyOS版）

**目标用户**：摄影爱好者

**核心定位**：综合性摄影平台

**开发框架**：ArkTS + ArkUI（HarmonyOS原生）

---

## 1. 整体架构

### 架构方案
模块化单体架构

### 应用架构分层

```
┌─────────────────────────────────────────┐
│         展示层 (Presentation Layer)      │
│  ┌──────┬──────┬──────┬──────┬──────┐   │
│  │照片  │社区  │约拍  │学习  │工具  │   │
│  │管理  │交流  │匹配  │中心  │箱    │   │
│  └──────┴──────┴──────┴──────┴──────┘   │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│         业务逻辑层 (Business Layer)      │
│  照片服务│社区服务│约拍服务│学习服务│...  │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│         数据层 (Data Layer)              │
│  本地数据库│云存储│缓存│文件系统        │
└─────────────────────────────────────────┘
```

### 技术栈
- **前端**：ArkTS + ArkUI（HarmonyOS原生）
- **本地存储**：RDB（关系型数据库） + Preferences
- **云服务**：华为云服务（认证、数据库、存储）
- **架构模式**：MVVM（Model-View-ViewModel）

---

## 2. 核心功能模块

### 2.0 用户管理模块
- **注册登录**：手机号/邮箱注册，支持华为账号一键登录
- **个人资料**：头像、昵称、简介、摄影风格标签
- **角色切换**：摄影师/模特双重身份，可随时切换
- **实名认证**：可选实名认证，提升可信度
- **隐私设置**：谁可以查看我的作品/联系我
- **账号安全**：密码修改、设备管理、注销账号

### 2.1 照片管理模块
- **相册管理**：创建/编辑/删除相册，支持自定义封面
- **智能分类**：按时间、地点、标签自动分类
- **云端同步**：照片自动备份到云端，支持多设备访问
- **批量操作**：批量移动、删除、分享

### 2.2 社区交流模块
- **作品广场**：瀑布流展示热门作品，支持点赞/收藏/评论
- **话题圈子**：摄影主题讨论区（风光/人像/街拍等）
- **关注系统**：关注摄影师，查看动态feed
- **消息通知**：评论/点赞/关注实时通知

### 2.3 约拍匹配模块
- **用户档案**：摄影师/模特资料展示（作品集/风格标签）
- **智能推荐**：基于位置、风格、评分匹配
- **约拍发布**：发布约拍需求（时间/地点/风格/费用）
- **沟通预约**：内置聊天，确认约拍详情

### 2.4 学习中心模块
- **视频教程**：分类视频课程（基础/进阶/专项）
- **技巧文章**：图文教程，参数解析
- **作品赏析**：大师作品+参数+点评

### 2.5 拍摄工具箱
- **参数计算**：景深计算器、曝光计算、超焦距计算
- **构图辅助**：九宫格、黄金螺旋、对角线
- **光线提醒**：黄金时段/蓝调时刻提醒

## 4. 界面设计

### 主要页面结构

```
Tab导航（底部）
├── 首页（社区动态流）
├── 照片（我的相册）
├── 约拍（匹配列表）
├── 学习（课程列表）
└── 我的（个人中心）
```

### 核心页面

| 页面 | 布局 | 关键元素 |
|------|------|----------|
| 首页 | 瀑布流 | 作品卡片、话题标签、搜索栏 |
| 照片管理 | 网格+列表 | 相册封面、照片缩略图、批量操作栏 |
| 约拍匹配 | 卡片列表 | 用户卡片（头像/作品数/评分）、筛选器、地图模式 |
| 学习中心 | 分类列表 | 课程卡片、进度条、收藏按钮 |
| 工具箱 | 图标网格 | 工具入口、最近使用 |
| 个人中心 | 列表 | 统计数据、功能入口、设置项 |

### 交互特点
- **手势操作**：照片支持双指缩放、左右滑动切换
- **下拉刷新**：所有列表页支持下拉刷新
- **快速操作**：长按照片弹出操作菜单

## 5. 数据流与同步策略

### 数据流向

```
用户操作 → ViewModel → Service → Repository
                              ↓
                    ┌─────────┴─────────┐
                    ↓                   ↓
              本地数据库(RDB)       云端API
                    ↓                   ↓
              缓存更新            云端同步
                    ↓                   ↓
              UI刷新 ← ViewModel ← 响应数据
```

### 同步策略

| 场景 | 策略 |
|------|------|
| 照片上传 | 先存本地，后台静默上传，失败自动重试 |
| 社区内容 | 优先加载缓存，后台拉取最新，增量更新 |
| 约拍信息 | 实时同步，WebSocket推送状态变更 |
| 学习进度 | 本地记录+云端同步，断网可继续学习 |

### 离线支持
- 核心功能支持离线使用（查看本地照片、使用工具）
- 网络恢复后自动同步待上传内容

## 6. 错误处理与安全性

### 错误处理策略

| 错误类型 | 处理方式 |
|----------|----------|
| 网络异常 | Toast提示 + 重试按钮 + 自动重连 |
| 上传失败 | 本地暂存 + 后台重试（最多3次） |
| 数据加载失败 | 骨架屏 + 空状态提示 + 刷新按钮 |
| 权限不足 | 引导用户授权 + 功能降级 |
| 服务器错误 | 友好提示 + 自动上报日志 |

### 安全性设计

| 方面 | 措施 |
|------|------|
| 用户认证 | JWT Token + 刷新机制 |
| 数据传输 | HTTPS加密 |
| 敏感信息 | 本地加密存储（密钥不落盘） |
| 图片防盗 | 云端图片带签名URL，有时效限制 |
| 内容审核 | 用户发布内容接入AI审核 + 人工复核 |
| 隐私保护 | 位置信息脱敏，联系方式仅约拍双方可见 |

## 7. 测试策略

### 测试层次

| 测试类型 | 范围 | 工具/方法 |
|----------|------|-----------|
| 单元测试 | ViewModel、Service、工具函数 | Jest |
| 组件测试 | ArkUI组件交互 | HarmonyOS测试框架 |
| 集成测试 | 模块间数据流、API调用 | 自动化测试脚本 |
| UI测试 | 页面跳转、手势操作 | 自动化UI测试 |
| 性能测试 | 启动速度、列表滚动、图片加载 | Profiler工具 |

- 核心业务逻辑：≥80%
- 工具类函数：≥90%
- UI组件：关键路径100%
